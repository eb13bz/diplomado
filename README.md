# proyecto diplomado

### 1. clonar el proyecto

```bash

git clone https://github.com/eb13bz/diplomado.git
```
### 2. instalar dependencias
```bash
npm install 
```

### 3. copiar variables de enterono y configurar 
```bash
cp .env.sample .env 
```
### 4. levantar el poyecto
#### 4.1 en desarrollo
```bash 
npm run dev 
```
#### 4.1 en produccion
```bash 
npm run start
```
